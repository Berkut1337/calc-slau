Массив значений - массив, в котором хранятся ненулевые значения взятые подряд из первой строки, затем идут значения из следующей строки и т.д.

Массив индексов столбцов - массив, который хранит номера столбцов, соответствующих элементов из массива значений.Массив начинается с нуля.

62 Массив индексации строк - массив, который для индекса i хранит количество ненулевых элементов в строках с первой до i-1 строки включительно, стоит отметить что последний элемент массива индексации строк совпадает с количеством элементов в матрице, а первый всегда равен 0.
